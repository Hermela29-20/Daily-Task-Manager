# daily_task_manager

A simple Flutter application that performs CRUD (Create, Read, Update, Delete) operations using a public REST API with Provider state management and the HTTP package.

#  Features

-  Fetch tasks from API (Read)
-  Add new task (Create)
-  Update task (Update)
-  Delete task (Delete)
-  Loading states
-  Error handling
